### 🏃‍♂️ UI/UX Design: Summer Splash Community Fun Run 2026

A promotional UI/UX card design created for **Team GFE's** inaugural, free community event. The design focuses on a vibrant, energetic, and refreshing summer theme to maximize community engagement.

*   **Project Type:** UI/UX Design / Event Promotion Card
*   **Event Title:** Summer Splash Community Run 2026 (Run, Refresh, Celebrate)
*   **Target Audience:** General Community / Fitness & Fun Enthusiasts
*   **Key Highlights:** 
    *   Features an eye-catching, water-themed header illustration showcasing active community participants.
    *   Optimized for social feeds with clear typography, essential event details, and direct calls-to-action.
